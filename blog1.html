How to deploy a website in github (9 STEPS)

STEP 1

You have to open github.com

STEP 2

You have to login to github or create a new github account

STEP 3

Click on + button in the top right header

Open image

STEP 4

There will be many option click on new repository

Open image

STEP 5

Enter the detail like the title and description. It is defaultly public you can choose your option public or private and please check the readme file

Open image

STEP 6

After creating a repository click on add file and select upload a file or create a new file

Open image

STEP 7

After upload all your files go to setting

Open image

STEP 8

In the setting there is a option name(Pages settings now has its own dedicated tab! Check it out here!) in the bottom of the github setting. click on Check it out here!

Open image

STEP 9

There will be two button branch and save click on branch and select main option if select main it will show to the /root is defaultly click on save. After some seconds there will be a dialog box that tells your site is ready to publish on https://user_name.github.io/project_name click on that And you site will be ready in 1 minutes or 2 minute

Open image

FINISH

Congratulation you have deployed a website in github
