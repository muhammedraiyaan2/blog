Learn how to use Visual Studio Code

If you have not installed Visual Studio Code go to this link



1. Changing the theme: If you want to change the theme go to settings and click on color theme and select your theme or press
Ctrl+k Ctrl+T

2. Opening a folder: If you want to open a folder then click on file and click on open folder or press Ctrl+O

3. Create a new file in a folder: click on + button and name the file and your file will be created after writing some code press Ctrl+S to save the file

4. Extension: extensions are used to make your IDE look better and help in many ways like theme, thunder client and Ssh remote etc. Go to extensions or presss Ctrl+Shift+X in your vs code and search some of extensions name. If you want to want more themes and the icon of the vs code files should be good then install this extension Material theme icon and Community Material theme. Material theme icon is used for making your file iocns look better and Community Material theme is used gives you many themes dark and light etc. After you install the extensions if the extension is not working please reopen your vs code.

5. Change keyboard shortcuts: Click on setting and click on Keyboard Shortcuts or press Ctrl+K Ctrl+S and search what you want to change like I want to change the theme shortcut to Alt+C instead of Ctrl+K Ctrl+T search in keyboard shortcuts theme and click on the pencil icon this icon only appear when you hover on the shortcut. And press the key that you want to change the keyboard shortcut and press Enter.

6. Settings: You can change the setting in the vs code you have to go in setting and click on setting or press Ctrl+, and search the setting and it will show you the setting. Like you can change the zoom of the vs code in mouse like Ctrl+MouseWheel

7. Creating a folder: To created folder follow the same method as you followed in how to create a new file in a opened folder. Now please select the folder button to create folder not file button in vs code expolorer.
