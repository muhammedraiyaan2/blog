How to make a http server in python (3 STEPS)



Steps:

1. Go to your folder where you want to start the server in the address bar type powershell or cmd and press enter and then the terminal will open on the folder

2. You have to type python -m http.server and press enter

3. Open the browser and type localhost:8000. Please don't exit the terminal if the work is finished then you can exit the terminal
