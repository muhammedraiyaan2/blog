15 Keyboard shortcuts for windows





Shortcuts:


1. Windows+D This Keyboard shortcut will go to desktop it minimize (Works for any windows).


2. AltTtab This Keyboard shortcut will switch programs (Works for any windows).


3. 4. Windows+L This keyboard shortcut will lock the computer and you have to unlock when ever you want to unlock (Works for any windows).


5. Windows+E This keyboard shortcut will open file explorer (Works for any windows).


6. Windows+number In this keyboard shortcut you have press windows + any number from 0 to 9 what happens in this shortcut is example see in the taskbar first program shortcut is windows+1 second program in the taskbar is window+2 etc (Works for any windows).


7. Windows+S This keyboard shortcut search anything that you want to search (Works only windows 10).


8. Windows+V When you press this keyboard shortcut a dialog box will open and it say that could you want to allow the history of the clipboard click on and you can able to see the copied clipboard history (Works for only windows 10).


9. Windows+R This keyboard shortcut will open run command (Works for any windows).


10. Windows+X This keyboard shortcut will open dialog box that shows show option like terminal or powershell run etc (For windows 10). For windows 7 or 8 when press the keyboard shortcut you can see a dialog box that shows battery brightness etc (For any windows not for windows 10).


11. Windows+P This keyboard shortcut helps you to connect multiply screen (For any windows).


12. Ctrl+Shift+Esc This keyboard shortcut open taskbar manager (For any windows).


13. Windows+PrtSc This keyboard shortcut will take a screenshot and save the screenshots in pictures (For only windows 10).


14. Windows+A This keyboard shortcut will show you some option (for only windows 10).


15. Windows+H In this keyboard shortcut A dialog box will be open where you can say something and it will type for you (For only windows 10).
